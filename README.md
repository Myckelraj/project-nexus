# project-nexus
# Welcome to the Simple Restaurant Website project. This project demonstrates a basic implementation of login and signup authentication. The goal is to create an attractive and simple restaurant website with a working authentication system.

## Features

- User signup with username, email, and password.
- User login with email and password.
- Attractive and simple user interface.
- Responsive design for various screen sizes.

## Technologies Used

- **Frontend:**
  - HTML5
  - CSS3
  - JavaScript
- **Backend:**
  - Node.js
  - Express.js
- **Other Tools:**
  - Git for version control
  - GitHub for repository hosting
